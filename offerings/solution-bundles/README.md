# Solution Bundles

This directory contains canonical messaging for sold, fixed-scope commercial bundles.

## Position in the repository

- `offerings/` describes core service offerings (Support, ExpertOps, Consulting)
- `offerings/solution-bundles/` describes packaged, sold combinations of those offerings
- `use-cases-value-pillars/` captures broader customer scenarios and shared value narratives that may map to one or more bundles

## Structure

All sold bundles are documented in the **bundle matrix** in `messaging.md` (intro plus one row per bundle: buyer moment, outcomes, inclusions). Scope detail, PS module mapping, and packaging variants belong in commercial systems (for example the PS Scope Book), not in separate files in this directory.

## Files

- `messaging.md`: canonical bundle messaging and bundle matrix
