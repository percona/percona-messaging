# About this repository

Percona Messaging is where approved company, product, and offering language lives before it is adapted into web pages, decks, campaigns, and enablement.

## What you are reading

This site is a **reading view** of the same markdown files contributors edit in GitHub. It is not a separate CMS. If you see a line here, it is the canonical version in the repository unless a downstream team has not yet updated their asset.

## The operating playbook

The full program guide explains why the repository exists, how teams work together, and how changes flow from intake to review to propagation:

**[Open source messaging playbook](/OPEN-SOURCE-MESSAGING-PLAYBOOK.md)**

That document is written for program participants as well as curious readers. You do not need to read it to find product or positioning copy; use the sidebar for that. It helps if you want context on governance, automation, and how messaging stays consistent over time.

## What this site does not replace

- Public product documentation at [docs.percona.com](https://docs.percona.com/)
- Marketing pages on [percona.com](https://www.percona.com/)
- Sales decks, battlecards, or campaign copy (those adapt from here)

When something ships externally, teams still hand off explicitly. This site makes the **in-repo source** easy to read and share.
